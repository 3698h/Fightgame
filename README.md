# 文字格斗游戏 - Java 学习练习项目

一个基于控制台的文字格斗游戏，使用 Java 面向对象编程实现。

## 项目简介

这是学习 Java 过程中完成的一个练习项目，通过实现一个完整的文字格斗游戏来巩固以下知识点：

- 面向对象编程（封装、继承、多态）
- 集合框架（ArrayList）
- 流程控制与用户输入处理
- 随机数生成
- 字符串处理

## 功能特性

### 登录注册系统
- 用户注册：用户名唯一性校验、长度/格式验证、手机号绑定
- 用户登录：验证码校验 + 密码验证（各3次机会）
- 忘记密码：通过手机号验证重置密码
- 账户锁定机制：密码输错3次自动锁定

### 角色系统
- 英雄角色：可自由分配20点属性（生命/攻击/防御），初始50蓝量
- 敌人角色：4种不同类型的敌人（战士、刺客、坦克、法师）
- 角色继承体系：`Character` → `HeroCharacter` / `EnemyCharacter`

### 战斗系统
- 回合制战斗，玩家可选择不同技能或使用道具
- 3种玩家技能：普通攻击、强力一击（消耗10MP）、生命汲取（消耗10MP回复0-20HP）
- 敌人拥有不同技能（猛击、快速攻击、防御姿态、火球术）
- 蓝条系统：释放技能消耗10MP，击败敌人恢复30%最大蓝量
- 防御状态伤害减半机制
- 伤害公式：`伤害 = 攻击力 - 防御力`（最低为1）
- 每次胜利后敌人属性增强（HP+10, ATK+3, DEF+2），同时玩家每3胜获得属性提升

### 道具系统
- 5种回血道具：桃子(+10)、煎蛋(+20)、花酿鸡(+30)、黑背鲈鱼(+40)、白玉汤(+50)
- 击败敌人后30%概率随机掉落道具
- 战斗中可选择使用道具回血

### 嘲讽系统
- 玩家被击败后，敌人会随机输出嘲讽话术

## 项目结构

```
Fightgame/
├── APP.java                    # 程序入口
├── domain/
│   ├── Character.java          # 角色基类（血量、蓝量、攻击、防御）
│   ├── HeroCharacter.java      # 英雄角色（技能列表、背包）
│   ├── EnemyCharacter.java     # 敌人角色（技能、防御状态）
│   ├── User.java               # 用户类（账号信息、手机号）
│   └── Consumable.java         # 道具类（恢复道具）
└── ui/
    ├── Login.java              # 登录注册界面
    └── FightingGame.java       # 游戏主逻辑
```

## 运行方式

1. 确保已安装 JDK 8+
2. 编译并运行 `APP.java`：

```bash
javac -d out src/Fightgame/**/*.java
java -cp out Fightgame.APP
```

或直接在 IntelliJ IDEA 中打开项目，运行 `APP.java` 的 `main` 方法。

## 游戏流程

```
启动 → 登录/注册 → 创建角色（分配属性点）→ 选择敌人战斗 → 回合制战斗 → 结算
                      ↑                                              ↓
                      └──────────── 继续战斗 / 退出游戏 ←─────────────┘
```

## 学习总结

通过这个项目练习了 Java 的核心语法和面向对象设计思想，包括类的继承与多态、集合的使用、Scanner 用户交互、随机数生成、字符串处理等基础知识。项目涵盖了登录注册、属性分配、回合制战斗、道具掉落等完整的游戏机制。
