# 文字格斗游戏 - Java 学习练习项目

一个基于控制台的文字格斗游戏，使用 Java 面向对象编程实现。

## 项目简介

这是学习 Java 过程中完成的一个练习项目，通过实现一个完整的文字格斗游戏来巩固以下知识点：

- 面向对象编程（封装、继承、多态）
- 集合框架（ArrayList）
- 流程控制与用户输入处理
- 随机数生成
- 字符串处理

## 功能特性

### 登录注册系统
- 用户注册：用户名唯一性校验、长度/格式验证
- 用户登录：验证码校验 + 密码验证（各3次机会）
- 账户锁定机制：密码输错3次自动锁定

### 角色系统
- 英雄角色：可自由分配20点属性（生命/攻击/防御）
- 敌人角色：4种不同类型的敌人（战士、刺客、坦克、法师）
- 角色继承体系：`Character` → `HeroCharacter` / `EnemyCharacter`

### 战斗系统
- 回合制战斗，玩家可选择不同技能
- 3种玩家技能：普通攻击、强力一击（消耗HP）、生命汲取（消耗HP回复）
- 敌人拥有不同技能（猛击、快速攻击、防御姿态、火球术）
- 防御状态伤害减半机制
- 伤害公式：`伤害 = 攻击力 - 防御力`（最低为1）
- 每3胜自动增强敌人属性，同时提升玩家属性

## 项目结构

```
Fightgame/
├── APP.java                    # 程序入口
├── domain/
│   ├── Character.java          # 角色基类（血量、攻击、防御）
│   ├── HeroCharacter.java      # 英雄角色（技能列表）
│   ├── EnemyCharacter.java     # 敌人角色（技能、防御状态）
│   └── User.java               # 用户类（账号信息）
└── ui/
    ├── Login.java              # 登录注册界面
    └── FightingGame.java       # 游戏主逻辑
```

## 运行方式

1. 确保已安装 JDK 8+
2. 编译并运行 `APP.java`：

```bash
javac -d out src/Fightgame/**/*.java
java -cp out Fightgame.APP
```

或直接在 IntelliJ IDEA 中打开项目，运行 `APP.java` 的 `main` 方法。

## 游戏流程

```
启动 → 登录/注册 → 创建角色（分配属性点）→ 选择敌人战斗 → 回合制战斗 → 结算
                      ↑                                              ↓
                      └──────────── 继续战斗 / 退出游戏 ←─────────────┘
```

## 学习总结

通过这个项目练习了 Java 的核心语法和面向对象设计思想，包括类的继承与多态、集合的使用、Scanner 用户交互等基础知识。
